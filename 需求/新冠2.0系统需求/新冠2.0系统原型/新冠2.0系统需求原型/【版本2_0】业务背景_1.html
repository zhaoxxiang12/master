功能用途：

数据统计是为了方便用户从统计层面了解新冠2.0系统数据，满足用户的数据统计需求而开发的功能；

数据统计是取新冠2.0系统中的上报数据，根据特定的条件进行统计并展示的功能；

前提条件：

有能够登录的管理账号

账号有数据统计模块的功能权限

管理端设置了上报工作量字段和统计字段；

下级上报机构上报了阳性案例和工作量的数据；

预期结果：

1、管理账号登录后有数据统计模块，在数据统计模块下有上报机构数统计、阳性案例数统计、上报工作量统计菜单，点击后可以进入上报机构数统计页面、阳性案例数统计页面、上报工作量统计页面；

2、在数据统计-上报机构数统计页面中，可以查看各级区域时间范围内每一天的上报机构数；

3、在数据统计-阳性案例数统计页面中，可以查看各级区域时间范围内每一天的阳性案例数和这段时间内的阳性案例数总和；

4、在数据统计-上报工作量页面中，可以查看各级区域时间范围内每一天的统计字段值和这段时间内的统计字段的总和；

权限规则说明：

数据统计模块下有上报机构数统计、阳性案例数统计、上报工作量统计这些功能点，分别对应上报机构数统计页面、阳性案例数统计页面、上报工作量统计页面。
