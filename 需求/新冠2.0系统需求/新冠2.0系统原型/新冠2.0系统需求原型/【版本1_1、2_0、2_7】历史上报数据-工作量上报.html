【历史上报数据】

 页面分页展示上报机构所有的历史上报记录，默认不展示任何数据，需通过“查询”才展示符合对应条件的数据，列表按照检测时间的先后顺序降序；

【查询】

1、支持检测日期查询，截止时间必须大于等于开始时间，时间精确到天，时间为必选项；

2、支持上报内容查询，上报内容以下拉框单选形式；选项为： b、全部工作量上报 c、工作量列表名（由管理端配置）【版本2.7】{现在不需要有以下选项：a、阳性案例 }

3、支持数据状态查询，数据状态为下拉框单选形式；选项为：a、未审核b、已审核c、已退审，默认为全部

1、上报数据由管理端审核，数据状态只有在退审的时候&在审核时间范围内，上报账号才可修改数据（修改按钮显示可点击状态），点击修改数据，弹出修改数据弹窗进行修改，修改弹窗表单的时间默认为上次上报的时间，可修改更新，不可选择未来时间

2、退审原因，只有有过退审操作才会有退审原因，退审原因显示在退审原因列中；【版本2.0】{所有退审原因按照顺序倒序显示，最新的退审原因显示在最上方}
