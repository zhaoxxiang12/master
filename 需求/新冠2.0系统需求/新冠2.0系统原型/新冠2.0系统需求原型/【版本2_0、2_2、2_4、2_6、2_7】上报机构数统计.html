【上报机构数页面概述】

页面展示管理机构下辖的区域中每一天符合条件的上报机构统计数；

页面是触发式，需要点击“查询”后才会显示数据；

0

1

查询项可展开和收起：

0、【版本2.2】{地区 - 目录树下拉框，默认选择当前管理机构区域以及下一级的行政区域，如果当前管理机构区域就是县级区域，那么就只显示本管理机构的区域；}

1、时间范围 - 时间控件 - 开始时间和结束时间查询，精确到天；

2、机构职能 - 下拉选择框 - 选项：采样机构、检测机构、采样和检测机构、非采样和检测机构，允许多选，默认为空；

3、机构类别 - 下拉选择框 - 选项：医疗、疾控、第三方实验室，允许多选，默认为空；

4、机构级别- 下拉选择框 - 选项与【机构类别】联动：当【机构类别】选择医疗时，显示三级甲等、三级乙等、三级、二级甲等、二级乙等、二级、一级医院、其他（或未定级）；当【机构类别】选择疾控时，显示省级、市级、县级；当选择第三方实验室时，显示第三方实验室，允许多选，默认为空；

5、机构性质 - 下拉选择框 -  选项：其他、中医医院、妇幼、专科、综合，允许多选，默认为空

6、隶属关系 - 下拉选择框 - 选项：其他、市属、省属、部属，允许多选，默认为空

7、经营性质 - 下拉选择框 - 选项：公立、私立、其他，允许多选，默认为空;

8、是否P2级实验室 - 下拉选择框 - 选项：是、否，允许多选，默认为空；

9、是否PCR备案 - 下拉选择框 - 选项：是、否，允许多选，默认为空；

10、是否新冠检测备案 - 下拉选择框 - 选项：是、否，允许多选，默认为空；

查询结构按照搜索条件显示对应地区的上报机构数量

单个筛选条件为多选，只要上报机构对应字段值是多选选项中的值，那么这个上报机构就为符合筛选条件的上报机构（比如机构职能选择了采样机构和检测机构，那么只要上报机构中的机构职能是采样机构或者检测机构的数据都为符合条件的数据）；

如果限制多个筛选条件那么需要同时符合这些限制筛选条件的上报机构才是符合筛选条件的上报机构（比如机构职能选择了采样机构和检测机构，机构类别选择了医疗和疾控，那么只有机构职能为采样机构或者检测机构，同时机构类别为医疗或者疾控的上报机构才是符合条件的数据）；

2

列表以目录树表格形式分级展示管理机构所在地区省-市-区/县，默认展开只展示下一级，页面进入后默认不显示统计数据，需要选择条件点击“查询”后页面才会显示对应的统计数据；

上报机构数统计逻辑：在管理账号的区域范围内的上报机构，在选择时间范围中每一天有上报数据（阳性案例和工作量，只要有一个上报就算已上报，这个机构纳入上报机构数统计）的上报机构数量的总和；

【版本2.2】{数据统计不限制时间，当数据有审核通过时，需要将对应时间的统计数据更新后缓存；

【版本2.4】{数据统计时的上报机构状态、阳性案例状态是按照上报时的上报机构状态、阳性案例状态为准，上报数据时是什么状态就记录为什么状态；}

只有审核通过的数据才会纳入统计；

地区作为筛选条件，可以选择全国以及全国各个省、市、县，允许同时选择全国、不同省、不同市、不同县的情况，选择后页面只展示对应地区的数据；当地区选择为空时，默认显示当前管理机构区域以及下一级管理机构区域的统计，比如全国账号登录，那么地区为空时显示全国以及各个省份的数据；

}

}
