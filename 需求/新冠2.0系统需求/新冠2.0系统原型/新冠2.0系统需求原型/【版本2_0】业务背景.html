功能用途：

消息互通设置是新冠2.0系统中保证数据上报、数据审核流程的正常进行、促进管理账号和上报账号的信息流通，而开发的消息互通的功能模块。

消息互通包括三个功能：1、提醒上报账号上报和管理账号审核的消息提醒功能；2、用于管理账号和上报账号的信息流通的公告板功能，公告板中又包括版本、操作手册、消息通知；3、调查问卷内容暂不展开；

前提条件：

有能够登录的管理账号

账号有消息互通模块的功能权限

需要先设置上报时间区间和审核时间区间，才能够使用消息提醒功能；

预期结果：

1、管理账号登录后有消息互通模块，在消息互通模块下有消息提醒设置菜单和消息提醒菜单，点击后可以进入消息提醒设置页面和消息提醒页面；

2、在消息互通-消息提醒设置页面中，可以添加上报提醒推送规则和审核提醒推送规则，可以在设置的时间点发送对应模板的短信；（时间点必须在上报时间区间或者审核时间区间范围内）

3、在消息互通-消息提醒页面可以点击按键给未上报的上报机构/未审核的审核机构中所有手机号发送对应模板的短信；（时间点必须在上报时间区间或者审核时间区间范围内）

4、管理账号登录后，在消息互通模块下有公告板菜单，公告板菜单下有“版本更新”、“操作手册”、“消息通知”三个菜单，点击后可以进入版本页面、操作手册页面、消息通知页面；

5、管理账号可以分别在版本更新页面、操作手册页面、消息通知页面中分别添加版本信息、操作手册信息、消息通知信息，并可以分别对版本信息、操作手册信息、消息通知信息进行维护；

6、在管理账号推送版本信息、操作手册信息、消息通知信息后，才会在对应的管理机构、上报机构中查看版本信息、操作手册信息、消息通知信息；

权限规则说明：

消息互通模块下有消息提醒设置、消息提醒、公告板-版本更新、公告板-操作手册、公告板-消息通知这些功能点，分别对应消息提醒设置页面、消息提醒页面、公告板-版本更新页面、公告板-操作手册页面、公告板-消息通知页面。
